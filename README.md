# Pizza_Sales_dashboard

Pizza Sales Analysis Project
This project involves analyzing a pizza sales dataset using SQL and Power BI. It aims to provide key business insights and performance metrics for a pizza store. The analysis includes:

Key Metrics:

Total Revenue

Average Order Value

Total Pizzas Sold

Total Orders

Average Pizzas per Order

Sales Trends:

Daily and Monthly Order Trends

Revenue Breakdown by Pizza Category and Size

Performance Insights:

Top 5 and Bottom 5 Pizzas by Revenue, Quantity Sold, and Number of Orders
